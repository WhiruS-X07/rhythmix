# 🎵 Rhythmix

Rhythmix is a modern web-based music player that allows users to discover, browse, and play music through the Jamendo API. The project focuses on delivering a clean and responsive user experience inspired by modern music streaming platforms.

## 🚀 Features

- 🎧 Music playback with audio controls
- 🔍 Search songs and artists
- ❤️ Add songs to favorites
- 🕒 Recently played tracks
- 📃 Queue management
- 🔀 Shuffle mode
- 🔁 Repeat mode
- 🎚 Volume control
- 📱 Responsive design
- 🌐 Dynamic music data using Jamendo API
- 💾 Local Storage support for user preferences

## 🛠 Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript (ES6+)
- Tailwind CSS

### API
- Jamendo Music API

### Browser Storage
- Local Storage

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/rhythmix.git
```

### 2. Open Project Folder

```bash
cd rhythmix
```

### 3. Run Project

Simply open:

```text
index.html
```

in your browser.

Or use VS Code Live Server:

```bash
Right Click → Open with Live Server
```

## 🔑 API Configuration

Rhythmix uses the Jamendo API.

Create your API key from:

```text
https://developer.jamendo.com/
```

Update the Client ID inside:

```javascript
api.js
```

```javascript
const CLIENT_ID = "YOUR_CLIENT_ID";
```

## 🎮 Controls

| Action | Function |
|----------|----------|
| Play/Pause | Start or pause playback |
| Next | Play next track |
| Previous | Play previous track |
| Shuffle | Random song playback |
| Repeat | Repeat current track |
| Search | Find songs or artists |
| Favorites | Save songs |
| Queue | Manage upcoming songs |
